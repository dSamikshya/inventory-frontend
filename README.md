# 📦 InvenPro - Inventory Management System

A modern, full-stack inventory management dashboard built with Angular and Spring Boot.

## 🚀 Tech Stack

**Frontend**
- Angular 19
- TypeScript
- CSS3

**Backend**
- Java Spring Boot
- REST API
- (Database coming soon)

## ✨ Features

- 📊 Real-time dashboard with stats cards (Total Items, Low Stock, Total Value, Categories)
- ➕ Add new inventory items via form
- 📋 View all items in a clean data table
- ✏️ Edit existing items
- 🗑️ Delete items
- 🟢 Stock status badges (In Stock / Low Stock)
- 🌙 Dark navy modern UI

## 🛠️ How to Run Locally

### Frontend

```bash
# Clone the repository
git clone https://github.com/dasmikshya/inventory-frontend.git

# Navigate into the project
cd inventory-frontend

# Install dependencies
npm install

# Run the development server
ng serve
```

Open your browser and go to `http://localhost:4200`
